# PhotoCours

Petit site pour aider une personne avec PAP / PPS a prendre ses cours en photo avec son telephone, puis a les envoyer par email pour impression ou photocopie.

## Utilisation

1. Ouvrir `index.html` dans un navigateur.
2. Renseigner l'email du destinataire.
3. Appuyer sur `Prendre ou ajouter des photos`.
4. Verifier les pages ajoutees.
5. Appuyer sur `Envoyer par email`.

Sur telephone, le site utilise le partage natif quand le navigateur le permet. Il faut ensuite choisir l'application email. Si le navigateur ne permet pas de joindre directement les photos, le site telecharge les images et ouvre un email avec une consigne pour les ajouter en pieces jointes.

## Important

Pour que le partage natif fonctionne au mieux sur telephone, le site doit etre ouvert depuis une adresse securisee en `https`.
